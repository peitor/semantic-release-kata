# Lift Button C#


## Run tests

To run the tests for the Lift Button C# project, follow these steps:

```shell
dotnet test
```

## Update version

To update the version of the Lift Button C# project, you can use the provided `updateVersion.sh` script. 

Run the script with the desired version number and the path to the `.csproj` file as arguments. For example:

```shell
sh updateVersion.sh 1.2.3 LiftButton/LiftButton.csproj
```

For more details, see the top-level [README.md](../README.md) file.
